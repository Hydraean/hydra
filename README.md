# hydra

> The Web Application for Project Seantinel Cloud-based Admin Dashboard

Install Dependencies:

**with yarn:**

```sh
yarn
```

Development:

```sh
yarn start
```

Build:

```
 yarn build
```

**with npm:**

Installation:

```sh
npm i
```

Development:

```sh
npm start
```

Build:

```
 npm build
```

Design System:
