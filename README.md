> This repository has a dedicated mirror for deployment:

https://github.com/quadroloop/hydra

Install Dependencies:

**with yarn:**

```sh
yarn
```

Development:

```sh
yarn start
```

Build:

```
 yarn build
```

**with npm:**

Installation:

```sh
npm i
```

Development:

```sh
npm start
```

Build:

```
 npm build
```
